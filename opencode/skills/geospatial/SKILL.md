## Backend

- PostGIS for spatial operations. If an operation can be done in a PostGIS query, we
prefer that. 
- Shapely for geometry manipulation
- GeoAlchemy2 for database integration

## Frontend

- MapLibre GL (svelte) for visualization
- Deck.gl for advanced geospatial visualizations

This application manages agricultural parcels with geospatial boundaries.
